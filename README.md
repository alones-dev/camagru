# Camagru

This project is an Instagram like web application that lets you create photo montages using a photo taken from your webcam, and share them with other users.
This project was developed with NextJS.


### Here is a list of the different features of the application
- Authentication system
- Changing account details (password, email, username)
- Integration and use of the webcam
- Photo editing from the photo with predefined images
- System for sharing images on a feed, likes and comments
